Downloading easyb

easyb is distributed as a archive containing an easyb jar file along with its associated dependencies (including Groovy). easyb plug-ins (like DbUnit) are packaged separately-- you can download them with an associated release here as well.

The current version of easyb is 0.9.6 (Sept 2009) and can be downloaded from easyb's project site at Google code. Note, there are different archives for downloading-- easyb itself and various associated projects like the IntelliJ plugin.

Older public releases can be found on easyb's Google code project site.

If you have any questions or comments, drop us a note on our mailing list.
